How to Use JTN IPTV Viewer on Windows

JTN IPTV Viewer is a Windows app for watching legal IPTV sources, free M3U playlists, and user-provided Xtream-style services.

Choose Your Download

There are two Windows packages:

  • Installer ZIP: best for most users. It adds Start Menu and Desktop shortcuts.
  • Portable ZIP: best if you do not want to install anything. Unzip it and run the app directly.

Install the App

  1. Download IptvViewer-Windows-Installer.zip.
  2. Right-click the ZIP and choose Extract All.
  3. Open the extracted folder.
  4. Double-click install.cmd.
  5. Launch IPTV Viewer from the Desktop or Start Menu shortcut.
If Windows shows a security warning, choose More info, then Run anyway only if you downloaded the app from a trusted source.

Use the Portable Version

  1. Download IptvViewer-Windows-Portable.zip.
  2. Extract the ZIP to a folder you can find again.
  3. Open the extracted folder.
  4. Double-click IptvViewer.exe.

To remove the portable version, close the app and delete the extracted folder.

First Launch

The app starts with a few free public playlist examples. These do not require a username or password.

  1. Select a free service from Saved Services.
  2. Click Refresh Live Only.
  3. Search or browse the channel list.
  4. Select a channel and press the play button.

Add Your Own Service

For an Xtream-style provider:

  1. Click Add under Saved Services.
  2. Enter an account name, server URL, username, and password.
  3. Click Save Service.
  4. Use Refresh Live Only, Refresh Movies Only, or Refresh Series Only.

For a direct M3U playlist:

  1. Click Add.
  2. Enter a name and the direct .m3u or .m3u8 URL.
  3. Leave username and password blank.
  4. Click Save Service, then use Refresh Live Only.

Favorites, History, and Backup

  • Use Add Favorite to save channels or videos you watch often.
  • Use Clear History if you want to remove watch history.
  • Use Export Saved Data to back up your saved services, favorites, and history.
  • Use Import Saved Data to restore settings on another install.

Where Data Is Stored

Your private app data is stored on your own Windows account under:

%LOCALAPPDATA%\IptvViewer

Public downloads do not include private paid IPTV accounts, passwords, favorites, history, guide data, or cached channel lists.

Important Use Note

JTN IPTV Viewer is for lawful IPTV sources only. The app does not provide private paid accounts, copyrighted streams, or hosted video content. Users are responsible for the playlist or provider sources they add.
